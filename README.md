# ACORP-S100L-Ethernet-Card-WOL-Mod
Easy hardware mod enables Wake on LAN support in acorp-s100l ethernet card
## Modification
Things Required:
- 15k ohm resistor SMD, 1 Piece
- Any rectifier diode like 1N4001/1N4007 or SMD, 1 Piece
